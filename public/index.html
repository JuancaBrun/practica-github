API RESTful

PIEZAS

GET /api/piezas Obtener todas las piezas.
GET /api/piezas/{id} Obtener una pieza.
POST /api/piezas Añadir una pieza.
UPDATE /api/piezas/{id} Modificar una pieza.
DELETE /api/piezas/{id} Eliminar una pieza.

VEHÍCULOS

GET /api/vehiculos Obtener todos los vehículos.
GET /api/vehiculos/{id} Obtener un vehículo.
POST /api/vehiculos Añadir un vehículo.
UPDATE /api/vehiculos/{id} Modificar un vehículo.
DELETE /api/vehiculos/{id} Eliminar un vehículo.
